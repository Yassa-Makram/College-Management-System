# College Management System

A desktop application built with **Java Swing**, **JDBC**, and **MySQL** to manage students, instructors, departments, and courses.

## Features

* Dashboard with multiple modules
* Student Management (CRUD)
* Department Management (CRUD)
* Instructor Management (CRUD)
* Course Management (CRUD)
* Search functionality
* Secure database queries using `PreparedStatement`
* Modern UI using **FlatLaf**

## Technologies Used

* Java
* Java Swing
* FlatLaf
* JDBC
* MySQL

## Getting Started

1. Start your MySQL server (XAMPP or MySQL).
2. Create the database and import the SQL script.
3. Update the database credentials in `DBConnection.java`.
4. Run `Main.java`.
